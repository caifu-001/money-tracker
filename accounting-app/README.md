# 记账助手 - 云同步版

一个简洁好用的记账应用，支持多端同步。

## 功能

- 💰 记账 - 收入/支出记录
- 📊 统计 - 支出分布图表
- 💳 预算 - 月度预算管理
- 📤 导出 - CSV/Excel 导出
- ☁️ 同步 - Supabase 云端同步

## 快速开始

### 本地运行

1. 克隆仓库
2. 直接在浏览器打开 `index.html`

### 部署到 Vercel

1. Fork 此仓库
2. 登录 [Vercel](https://vercel.com)
3. 导入此项目
4. Deploy！

### 部署到 GitHub Pages

1. Fork 此仓库
2. 进入 Settings → Pages
3. 选择 `main` 分支
4. 访问 `https://yourusername.github.io/accounting-app`

## 技术栈

- 原生 HTML/CSS/JavaScript（零依赖）
- Supabase（云数据库）

## 数据同步

应用使用 Supabase 作为后端数据库。数据存储在你的 Supabase 项目中，支持多设备同步。

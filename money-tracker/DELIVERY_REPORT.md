## 🚀 快速开始

### 本地开发
```bash
# 1. 安装依赖
npm install

# 2. 配置环境
cp .env.example .env.local
# 编辑 .env.local，填入 Supabase 信息

# 3. 启动开发
npm run dev
```

### 部署到 GitHub
```bash
# 1. 创建 GitHub 仓库
# 2. 推送代码
git push -u origin main

# 3. 配置 GitHub Secrets
# 4. 启用 GitHub Pages
```

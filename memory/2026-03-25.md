## Key Technical Details

### Permission Model
- **Regular Users**: 
  - Can only see own transactions
  - Can only see own analytics
  - Cannot access admin features
  
- **Admins**:
  - Can view all users' transactions in same ledger
  - Can view all users' analytics
  - Can approve/reject new users
  - Can assign/revoke admin roles
  - Can reset passwords
  - Can delete users

### Data Query Pattern
```typescript
// Regular user: filtered by user_id
query.eq('user_id', user?.id)

// Admin viewing specific user: filtered by selected user
query.eq('user_id', selectedUserId)

// Admin viewing all: no user filter
```

## 🚀 立即开始（3步）

### 第一步：安装依赖
```bash
cd money-tracker
npm install
```

### 第二步：配置环境
```bash
cp .env.example .env.local
# 编辑 .env.local，填入 Supabase 信息
```

### 第三步：启动开发
```bash
npm run dev
# 访问 http://localhost:5173
```
